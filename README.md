# Hospital Readmission Risk Dashboard

## Project Overview
Built an end-to-end healthcare analytics dashboard to identify
patients at high risk of 30-day readmission using data from
130 US hospitals (1999-2008).

## Tools Used
- Excel: Data exploration and quality audit
- Python (pandas): Data cleaning and feature engineering
- SQL (MySQL): Data storage and trend queries
- Power BI: Interactive dashboard and visualisation

## Key Findings
- Elderly patients (70+) show a 14.2% readmission rate
- Patients on 10+ medications have 2x the readmission risk
